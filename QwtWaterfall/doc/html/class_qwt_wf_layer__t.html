Detailed Description

Structure for layer definitions.

Layers, that are QImage in nature, are used to render some raster data.

Parameters
imageImage widget itself
wWidth of the layer
hHeight of the layer
minxMinimum x value for the layer
maxxMaximum x value for the layer
minyMinimum y value for the layer
maxyMaximum y value for the layer
XscaleAttached Qwt X scale
YscaleAttached Qwt Y scale
noscaleXSwitch on/off the scaling of the layer on X
noscaleYSwitch on/off the scaling of the layer on Y
